
# ➗ Math Quiz Game - C++ 

A console-based C++ application that generates random math questions with different difficulty levels and arithmetic operations. 

> ⚠️ This is a learning project created to practice C++ fundamentals, problem-solving, functions, structures, enums, and modular program design. 

--- 

## 📌 Features 

- Generate random math questions
- Multiple difficulty levels (Easy, Medium, Hard, Mixed)
- Multiple operation types (Addition, Subtraction, Multiplication, Division, Mixed)
- Calculate and validate player answers
- Display correct and incorrect answers
- Track right and wrong answers
- Show final quiz results
- Display pass/fail status
